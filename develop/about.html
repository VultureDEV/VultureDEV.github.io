About

VultureSMP is a private vanilla Minecraft Server

VultureSMP spawn town bridge, facing west.

Currently, our community consists of Loading... members. However, sometimes we run applications to join the server. The best way to keep track of this is to follow @VultureSMP on Twitter.

VultureSMP was founded by Loading... on October 28, 2014, and it is still going strong, with an amazing community.

If you feel so inclined, feel free to contact us!

We also have a number of members hosting let's plays on their personal YouTube channels, so for some action from the server, be sure to check out the official playlist of all videos!

On the server, we have some command block contraptions in effect. They are the one player sleep system by Xisuma, the anti-enderman griefing system by Xisuma, and the skull dropping system by Cowie
