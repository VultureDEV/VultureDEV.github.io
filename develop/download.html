How to Install

  • Once downloaded, open the ZIP folder
  • Copy the file folder world
  • For Windows, open Explorer, and go to the address %appdata%\.minecraft\saves
  • For Mac OS X, open Finder, and enter address ~/Library/Application Support/minecraft/saves (~ denotes home directory)
  • Paste the folder in this directory
  • The world should now show in the single player menu, you will have to reload the menu if you already had it open
