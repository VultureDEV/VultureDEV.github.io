How to install:
-Once downloaded, open the zip folder.
-Copy the file folder world.
-For Windows, open Explorer, and go to the address %appdata%\.minecraft\saves
-For Mac OS X, open the Finder, and enter address ~/Library/Application Support/minecraft/saves (~ denotes home directory)
-Paste the folder in this directory.
-The world should now show up in the single player menu, you will have to refresh the menu if you already had it open.
